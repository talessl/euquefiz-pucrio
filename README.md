# EuQueFiz

Este repositório contém a documentação e a prototipação inicial do projeto focado em gamificação culinária, visando o desenvolvimento de hábitos de economia e autonomia para jovens adultos.

## 🔗 Links Principais

*   **Showcase do Projeto:** [Assista à demonstração aqui](https://youtu.be/wSMfEsIz_UI) - Confira o vídeo de apresentação com a visão geral e o funcionamento da solução.
*   **Miro (Board de Planejamento):** [Acesse o quadro no Miro](https://miro.com/app/board/uXjVHIRSeLA=/?share_link_id=979609919522) - Explore o fluxo do produto, personas, jornada do usuário e o planejamento do MVP.

## 🛠 Sobre a Metodologia
Este projeto foi desenvolvido utilizando práticas ágeis, com foco em:
*   **MVP (Produto Mínimo Viável):** Focado na entrega de valor imediato com o menor escopo possível.
*   **UX Centrado no Usuário:** Desenvolvido para minimizar a fricção e acelerar o "momento Aha!" da economia doméstica.
*   **Backlog Estruturado:** Gestão de tarefas por prioridade e dependências técnicas.
